## Built With
- Zig master (at least `0.10.0-dev.531+3eb8d01f5`)
- See [`zig.mod`](./zig.mod) and [`zigmod.lock`](./zigmod.lock)

### Build from Source
Initially,
```
$ git clone https://github.com/nektro/zigmod --recursive
$ cd zigmod
$ zig build -Dbootstrap
$ ./zig-out/bin/zigmod fetch
```

To build,
```
$ zig build
$ ./zig-out/bin/zigmod
```
